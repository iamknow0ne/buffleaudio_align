one job, done clearly

Line up the stack while keeping the performance alive.

Put Align on the dub or backing vocal, route the lead as the Guide when your host supports it, and read a signed offset with confidence instead of guessing whether the double is early or late.

studio flow

A studio workflow for doubles, harmonies, rap stacks, and ADR.

01

Pick the role

Double Tight, Choir Natural, Rap Stack, and ADR Loose set musical starting points for tightness, naturalness, consonant control, guide blend, and stereo focus.

02

Trust the read

The Trust Meter names what Align hears: missing Guide, quiet lanes, unstable timing, locked timing, or a safe early/late Dub correction.

03

A/B the change

Original, Aligned, All Diff, and Tamer modes split the full timing change from consonant-only cleanup so the preview does not hide what changed.

04

Keep it human

Naturalness Guardrail flags risky cleanup, while Consonant Tamer Lite softens unmatched Dub attacks without flattening the vowel body.

inside the plugin

A transparent alignment tool, not a black-box warper.

Align focuses on the decisions producers actually make during vocal comp and stack cleanup: listen, trust the read, nudge carefully, tame clutter, and print only when the move still feels musical.

timingSigned Guide/Dub offset, confidence-gated suggestions, manual nudge, and delay-safe early/late correction in supported hosts.
toneConsonant Tamer Lite, Naturalness Guardrail, changed-material meter, All Diff audition, and Tamer-only audition.
sessionStack Role presets, phrase-health states, Next Best Move guidance, and Copy Report for sharing the current alignment read.
